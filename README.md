# user-management

1. Clonar el proyecto
2. ```yarn install```
3. Levantar la base de datos 
```
docker-compose up -d
```
4. Clonar el archivo ```.env.example``` y renombrarlo a ```.env``` 
5. Cambiar las variables de entorno(si es necesario)
6. Levantar: ```yarn start:dev```
